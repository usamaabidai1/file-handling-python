# file-handling-python

This Python code is demonstrating basic file handling operations, including reading, writing, and appending to text files, as well as working with JSON and CSV files. This project aims to provide examples and a reference for various common file operations in Python.

## Features

- Read from and write to text files
- Append content to existing text files
- Handle JSON data, including reading, writing, and filtering
- Read and write CSV files
- Filter CSV data and write to new files


## Setup

1. **Clone the repository:**

```bash
git clone https://github.com/usamaabidai1/FileOperations.git
cd FileOperations
```

2. **Ensure you have the necessary files:**

Create the required files (`my-text.txt`, `my_json.json`, `servershealth-220925-204801.csv`, etc.) in the same directory as the scripts.

## Usage

The project includes multiple scripts demonstrating various file operations. Ensure you have the necessary files in place and run the scripts to see the operations in action.

## Files

- `file_handling_python.py`: Main script demonstrating file operations
